PORTFOLIO DIGITAL PRIMA AJI SETYAWAN
GEMINI ARENA PRIMA

Saya Prima Aji Setyawan. Saya mahasiswa semester 6 D4 Teknologi Rekayasa Elektronika di Politeknik Negeri Semarang. Repositori ini menyimpan kode sumber portofolio digital saya.

TENTANG SAYA DAN REPOSITORI INI
Saya fokus pada pengembangan IoT dan otomatisasi industri. Saya sering menggunakan Gemini dan GitHub Copilot untuk mempercepat penulisan kode dan logika program. Situs portofolio ini adalah aplikasi halaman tunggal. Saya merakitnya menggunakan React dan Tailwind CSS. Kamu bisa mengeksplorasi implementasi kursor kustom, efek visual matriks, dan mode terang korporat di dalam kode.

PROYEK UNGGULAN
1. Post Stroke Smarthand Gloves. Saya memimpin pemrograman perangkat ini menggunakan ESP32 S3 dan antarmuka LVGL. Inovasi ini memenangkan Medali Perak pada pameran IIIEX 2025.
2. Water Level Control PID. Saya menerapkan algoritma PID presisi tinggi pada Arduino Nano dengan sensor Lidar. Sistem ini menekan tingkat kesalahan fluktuasi air hingga di bawah 2 persen.
3. FirebaseMicrocontroller. Saya mengintegrasikan modul ESP8266 untuk mengirim data sensor suhu ke Google Firestore secara waktu nyata.
4. Smartgate RFID System. Saya menghubungkan pembaca RFID dan motor servo dengan pangkalan data Firebase untuk sistem keamanan gerbang.
