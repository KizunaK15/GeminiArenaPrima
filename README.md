GEMINI ARENA PRIMA
PORTFOLIO DIGITAL PRIMA AJI SETYAWAN

Saya Prima Aji Setyawan. Saya mahasiswa semester 6 program D4 Teknologi Rekayasa Elektronika di Politeknik Negeri Semarang. Ini adalah kode sumber untuk portofolio digital saya. Saya merancang situs web ini sebagai aplikasi halaman tunggal. Saya menggunakan React dan Tailwind CSS. Situs ini menampilkan proyek proyek saya sebagai pengembang IoT dan sistem tertanam.

FITUR UTAMA APLIKASI
1. Saya menyediakan tema gelap bergaya matriks siber dan tema terang korporat.
2. Kamu bisa mengganti bahasa antara Indonesia dan Inggris.
3. Saya membuat kursor kustom interaktif untuk melacak pergerakan mouse.
4. Saya menambahkan elemen visual seperti efek tulisan rusak dan kartu miring tiga dimensi.
5. Saya menerapkan desain responsif untuk menyesuaikan tampilan pada layar komputer dan ponsel.

TEKNOLOGI PENGEMBANGAN
1. React JS versi 18
2. Babel untuk kompilasi JSX
3. Tailwind CSS
4. HTML5

PROYEK UTAMA SAYA
1. Water Level Control PID. Saya menggunakan Arduino Nano dan sensor Lidar untuk menstabilkan fluktuasi air. Tingkat kesalahan sistem ini di bawah 2 persen.
2. Post Stroke Smarthand. Saya mengembangkan sarung tangan interaktif berbasis ESP32 S3 dan antarmuka LVGL. Proyek ini mendapat Medali Perak pada pameran penemuan internasional IIIEX 2025.
3. PID Poultry Monitor. Saya membangun sistem kendali suhu dan kelembaban otomatis. Sistem ini secara aktif menurunkan angka kematian unggas.
4. Smartgate RFID System. Saya menggunakan modul RFID dan motor servo untuk menggerakkan pintu gerbang fisik. Saya mengintegrasikan perangkat keras ini dengan Firebase untuk mengelola akses keamanan secara waktu nyata.
5. Mushroom IoT Monitor. Saya merancang sistem penyemprotan air otomatis berbasis Blynk. Sistem ini menjaga suhu dan kelembaban ideal untuk kualitas panen jamur.
6. Stairs Climbing Robot. Saya menerapkan sistem kinematika dan mekanika khusus. Sistem ini memungkinkan robot menaiki tangga curam.
